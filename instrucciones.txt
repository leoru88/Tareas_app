PASOS PARA INSTALAR E INICIAR EL PROYECTO:
-----------------------------------------


1. Entrar al sitio web https://github.com/ e iniciar sesión.

2. Entrar al enlace https://github.com/leoru88/tareas_app.git y proceder a clonar el repositorio mediante URL HTTPS en el directorio deseado.

3. Crear una base de datos para el proyecto.

4. Abrir la carpeta del repositorio en el directorio escogido a través de un editor de código fuente.

5. Cambiar el nombre del archivo ubicado en la raíz del proyecto “.env.example” a “.env”.

6. Configurar las credenciales del manejador de base de datos en el archivo “.env” ubicado en la raíz del proyecto.

7.	Instalar:

	- PHP (versión utilizada 8.1.10)
	- Laravel.

8.	Instalar las siguientes dependencias:

	- Composer
	- NPM (instalar y actualizar)
	- Bootstrap
	- Axios
	- Vitejs Plugin
	- Laravel Mix

9.	Ejecutar los siguientes comandos:

	- Php artisan key:generate
	- Php artisan migrate
	- Npm run dev
	- Php artisan serve

10.	Abrir el navegador e ingresar a la URL del host. Ejemplo: http://127.0.0.1:8000/ para visualizar el proyecto.
